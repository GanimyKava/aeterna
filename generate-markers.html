Step 2: Download Markers

  1. In the marker generator, click "Generate" to create a new marker
  2. Download the marker pattern file (it will be a .patt file)
  3. Save it as pattern-marker1.patt in the markers/ folder
  4. Generate a second marker and save it as pattern-marker2.patt

Step 3: Print or Display Markers

  • Option A: Print the marker images and use them physically
  • Option B: Display markers on another device/tablet for testing
  • Option C: Use the marker images on your screen for demo purposes

Alternative: Use NFT Markers

If you prefer markerless AR using image tracking, you can use NFT (Natural Feature Tracking) markers. These work by recognizing images rather than specific patterns, but require more setup.

For now, pattern markers are recommended for easier setup.
